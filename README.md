# FFO-XGBoost for Microalgae Growth Prediction

This repository contains the dataset and Python code used in the study:
**"Prediction of Microalgae Growth Using Fennec Fox Optimization–XGBoost"**

## Contents
- `dataset.csv` — input features and microalgae growth data  
- `FFO-XGBoost_Microalgae_Growth_Prediction.py` — Python code implementing the hybrid FFO–XGBoost model  
- `README.txt` — variable descriptions
- Statistical description of the variables.docx — Statistical description of the variables
